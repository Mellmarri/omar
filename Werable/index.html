Practica 2: Introducción al Arduino II

Omar Ramírez de la Cruz
Ingeniería Mecatrónica
Fecha: 30 de Agosto del 2023
Arduino Electrónica

- Resumen -

En la practrica 1 Vamos a realizar 17 diferentes actividades con el arduino, desde ver el funcionamiento de la placa con los leds que contiene el mismo led hasta tener inputs y outputs como leds y botones, no son practicas complicadas pero más que enseñarnos al uso del arduino viene siendo un repaso del uso basico del arduino.

- Introducción -

En esta practica utilizaremos el Arduino Uno, con este y ayuda de otros materiales podremos hacer diferentes circuitos, simples, de no más de 5 elementos conectados al mismo tiempo, pero con la suficiente complicidad para entender el uso del arduino no solo como un elemento de conexion sino tambien de programación, puesto que diferentes codigo puedes terminar haciendo lo mismo de manera más o menos eficiente, eso es lo que se va a ver en las practicas siguientes.

Con lo realizado en esta practicas ya tenemos una base de conocimiento sobre el arduino, el como conectar los componentes, ver los inputs y outputs, y el como programar los componentes que conectamos para que hagan lo que se requiere o busca hacer.

- Materiales -

-Arduino Uno

-Protoboard

-4 LEDS (no importa el color)

-6 resistencias de 220 ohms

-2 Push Button

-Cables dupoint macho-macho (tambien conocidos como jumpers)

- Desarrollo -

Para el desarollo de las practicas se va a comentar individualmente las 17 practicas en forma de listado siguiendo el orden de las practicas.

~ Practica 1 ~

Para la practica 1 solo necesitaremos el Arduino Uno, la practica 1 consiste en encender el led que contiene la mis placa del Arduino Uno

Programa

para la programación tenemos sulamente un pinMode, siendo en del arduino, al cual simplemente le estamos dando el valor de HIGH para que se mantenga encendido, el codigo es el siguiente:

void setup()

{

pinMode(LED_BUILTIN, OUTPUT);

}

void loop()

{

digitalWrite(LED_BUILTIN, HIGH);

}

Diseño fisico

Para las conexiones fisicas simplemente conectamos el Arduino Uno con la computadora a travez de su cable USB para que se suba el codigo y el LED de la placa se encienda como se asegura el codigo.
