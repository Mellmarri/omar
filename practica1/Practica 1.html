Arduino Electrónica

- Resumen -

En la practrica 1 Vamos a realizar 17 diferentes actividades con el arduino, desde ver el funcionamiento de la placa con los leds que contiene el mismo led hasta tener inputs y outputs como leds y botones, no son practicas complicadas pero más que enseñarnos al uso del arduino viene siendo un repaso del uso basico del arduino.

- Introducción -

En esta practica utilizaremos el Arduino Uno, con este y ayuda de otros materiales podremos hacer diferentes circuitos, simples, de no más de 5 elementos conectados al mismo tiempo, pero con la suficiente complicidad para entender el uso del arduino no solo como un elemento de conexion sino tambien de programación, puesto que diferentes codigo puedes terminar haciendo lo mismo de manera más o menos eficiente, eso es lo que se va a ver en las practicas siguientes.

Con lo realizado en esta practicas ya tenemos una base de conocimiento sobre el arduino, el como conectar los componentes, ver los inputs y outputs, y el como programar los componentes que conectamos para que hagan lo que se requiere o busca hacer.

- Materiales -

-Arduino Uno

-Protoboard

-4 LEDS (no importa el color)

-6 resistencias de 220 ohms

-2 Push Button

-Cables dupoint macho-macho (tambien conocidos como jumpers)

-1 Display de 7 segmentos

Para el desarollo de las practicas se va a comentar individualmente las 17 practicas en forma de listado siguiendo el orden de las practicas.

~ Practica 1 ~

Para la practica 1 solo necesitaremos el Arduino Uno, la practica 1 consiste en encender el led que contiene la mis placa del Arduino Uno

Programa

para la programación tenemos solamente un pinMode, siendo en del arduino, al cual simplemente le estamos dando el valor de HIGH para que se mantenga encendido, el codigo es el siguiente:

void setup()

{

pinMode(LED_BUILTIN, OUTPUT);

}

void loop()

{

digitalWrite(LED_BUILTIN, HIGH);

}

Diseño fisico

Para las conexiones fisicas simplemente conectamos el Arduino Uno con la computadora a travez de su cable USB para que se suba el codigo y el LED de la placa se encienda como se asegura el codigo.

Construcción

No se utilizaron cables dupoint ni una protoboard en este caso, solamente hubo conexion usb a la placa de Arduino Uno.

Resultados

Se puede ver como el led del arduino permanece encendido, es el que se encuentra en la parte izquierda superior con la simbología de L a su costado.

Ahora en vez de ser el led de a placa Arduino el que se encienda lo hace el led que conectamos a los puertos, puesto que hemos dado la señal de que el pinmode 13 en este caso sea el que tenga señales de HIGH y LOW haciendo que el LED se encuentre parpadeando.

En la plactica 6 utilizamos cables dupoint, la protoboard, un LED, una resistencia y el Arduin Uno para seguir haciendo parpadear el led pero ahora con conexiones en la porotboard par aprneder su funcionamiento.

~ Practica 9 ~

Para la Practica 9 utilizaremos 2 leds, 2 resistencias, 1 protoboard, cables dupoint y el Arduino Uno, ahora haremos parpadear a dos leds consecutivamente.

Programa

para la programación ahora contamos con dos pinMode, uno para Led, a los que les daremos valores de HIGH y LOW para que parpaden pero de manera invertida, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

pinMode(12, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

digitalWrite(12, LOW);

delay(1000);

digitalWrite(13, LOW);

digitalWrite(12, HIGH);

delay(1000);

}

Diseño fisico

La única diferencia que encontramos en esta conexión es que ahora cada led tiene un pin diferente, sinedo el 13 y 12 respectivamente.

Construcción

La conexion es similar a la anterior, la diferencia es que ahora en vez de conectar las dos resistencias al pin13 ahora a cada resistencia le llega un pin diferente, siendo el 12 y el 13, puesto que ahora cada led hace una acción diferente.

Resultados

Podemos ver como ahora en vez de que se enciendan los dos leds simultaneamente ahora lo hacen de manera consecutiva, cuando uno se apaga el otro se enciende, como luces de navidad o de policia.

~ Practica 12 ~

Para la practica 12 utilizaremos la protoboard, la Placa Arduino Uno, cables dupoint, un led, una resistencia y un push button, ahora encenderemos leds con botones.

Programa

Este codigo no es tan largo así que lo pongo, ahora tenemos una variable de salida (LED) y una de entrada (pushh button), simplemente cuando el push button sea precionado dara corriente al led, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

pinMode(8, INPUT);

}

void loop()

{

digitalWrite(13, digitalRead(8));

}

Diseño fisico

la conexion de este codigo es medianamente simple, usando cables supoint damos fuente al led y al arduino formando una especie de circuito cerrado.

Construcción

El pin de entrada va conectada al lado negativo del botton, junto a la resistencia que llega a tierra, por el otro lado tiene la conexion al voltaje, por su parte el led igualmente se conecta como lo tuvimoa antes, el pin de entrada a la resistencia y la resistencia al catodo del del, el anodo del led a la tierra y es todo.

Resultados

Podemos ver en el video que el led se enciende cuanod se presiona el boton y solamente permanece encendido si el boton igualmente se mantiene prendido, como si el boton fuese el que cerrara el circuito..
