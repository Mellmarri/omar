- Resumen -

En la practrica 2 vamos a realizar 7 pequeñas practicas con las que seguimos aprendiendo del arduino pero ahora utilizando valores analogicos y digitales, esto ya utilizando diferentes herrameintas que botones y leds, introduciendo a la ecuacion los servomotores.

- Introducción -

Una señal analogica es una amgnitud que puede tomar cualquier valor dentro de un intervalo negativo y positivo.

Con las prácticas que realizaremos vamos a ver como funciona un servomotor y ver en la ultima práctica las señales analogicas usando una fuente de corriente alterna.

- Materiales -

-Arduino Uno

-Protoboard

-2 LEDS (no importa el color)

-2 resistencias de 220 ohms

-2 servomotores

-2 voltimetros

-Fuente de corriente alterna

-Cables dupoint macho-macho (tambien conocidos como jumpers)

~ Practica 3 ~

Para la practica 3 solo necesitaremos el Arduino Uno, la practica 3 consiste hacer parpadear el LED que contiene la placa Arduino Uno.

Programa

para la programación tenemos solamente un pinMode, siendo en del arduino, al cual simplemente le estamos dando el valor de HIGH un delay intermedio y el valor de LOW para obtener el parpadeo del LED, el codigo es el siguiente:

void setup()

{

pinMode(LED_BUILTIN, OUTPUT);

}

void loop()

{

digitalWrite(LED_BUILTIN, HIGH);

delay(100);

digitalWrite(LED_BUILTIN, LOW);

delay(100);

}

Diseño fisico

Para las conexiones fisicas simplemente conectamos el Arduino Uno con la computadora a travez de su cable USB para que se suba el codigo y el LED de la placa se encienda y apague como se asegura el codigo.

Construcción

No se utilizaron cables dupoint ni una protoboard en este caso, solamente hubo conexion usb a la placa de Arduino Uno.

Resultados

Podemos apreciar como el led parpadea con un retraso de milisegundos pero siendo el tiempo suficiente para que se note el parpadeo del LED de la placa Arduino Uno.

~ Practica 5 ~

En la practica 5 utilizaremos un elemento externo a la placa de Arduino Uno, siendo este un LED y una resistencia para evitar quemarlo.

Programa

para la programación tenemos solamente un pinMode, ahora con valor del puerto de la placa Arduino Uno, al cual simplemente le estamos dando el valor de HIGH un delay intermedio y el valor de LOW para obtener el parpadeo del LED, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

delay(1000);

digitalWrite(13, LOW);

delay(1000);

}

Diseño fisico

Para las conexiones fisicas Conectamos una resistencia al gnd de la placa Arduino, que conectara con el Led para que de esta forma llegue las señales del parpadeo.

Construcción

Para la construcción del circuito de esta practica conectaremos la resistencia al pinmode y esta resistenica ira direcmantente al led que cerrara el circuito para que el LED encienda, de hehco la imagen de referencia esta mal puesto que al no tener resistencia lo más posible es que ese led este quemado.

Resultados

Ahora en vez de ser el led de a placa Arduino el que se encienda lo hace el led que conectamos a los puertos, puesto que hemos dado la señal de que el pinmode 13 en este caso sea el que tenga señales de HIGH y LOW haciendo que el LED se encuentre parpadeando.

~ Practica 6 ~

En la plactica 6 utilizamos cables dupoint, la protoboard, un LED, una resistencia y el Arduin Uno para seguir haciendo parpadear el led pero ahora con conexiones en la porotboard par aprneder su funcionamiento.

Programa

para la programación seguimos teniendo solamente un pinMode, seguimos con el led parpadeante (pinmode 13), al cual simplemente le estamos dando el valor de HIGH un delay intermedio y el valor de LOW para obtener el parpadeo del LED, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

delay(1000);

digitalWrite(13, LOW);

delay(1000);

}

Diseño fisico

Introduciendo una de las herramientas más importantes y que más vamos a usar, la protoboard entra en la practica junto a los cables dupoint, conectaremos todavia directamente el circuito, con ayuda de los cables dupoint macho -macho, a una resistencia y led para que logre llegar la informacion.

Construcción

la Protoboard nos ayuda a conectar elementos de forma vertical en los centros y horizontal en las cargas, de forma que conectamos verticalmente el pin13 a la resistencia, la resistencia al catodo del led, el anodo del led al gnd del Arduino, como se aprecia en la imagen anterior.

Resultados

Podemos apreciar como el LED sigue parpadeando pero ahora gracias a las conexiones hechas en la protoboard, forma de aprender a usar la protoboard pero no completamente ya que en la siguiente plactica usaremos sus terminales de gnd y vcc.

~ Practica 7 ~

Para la practica 7 utilizaremos 1 led, 1 resistencia, cables dupoint, 1 protoboard y el Arduino Uno, en esta plactica haremos parpadear un LED siguiendo aprendiendo a usar la Protoboard

Programa

para la programación seguimos teniendo solamente un pinMode, seguimos con el led parpadeante (pinmode 13), al cual simplemente le estamos dando el valor de HIGH un delay intermedio y el valor de LOW para obtener el parpadeo del LED, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

delay(1000);

digitalWrite(13, LOW);

delay(1000);

}

Diseño fisico

Ahora utilizanod más cables dupoint, seguimos con el led y la resistencia conectada a la Protoboard pero ahora no estan conectado directamente, vemos como el gnd llega a una lina al igual que lo hace el vcc y de alli se conectan directamente al led, util si tenemos más de un componente que utilice el mismo pinmode como veremos más adelante .

Construcción

El pinmode se conecta directamente a la resistencia y la resistencia al LED como veniamos haciendo en la practica anterior pero ahora el gnd llega desde la misma protoboard y no directamente de la placa arduino, se hace una especie de puente que vemos como llega el gnd a linea de gnd y de alli el led obtiene su tierra, que sigue siendo proporcionada por el arduino pero ahora no es directa y esto nos sirve por si queremos tener más de una conexion que utilice el voltaje o la tierra.

Resultados

Seguimos apreciando el parapdeo del led pero ahora con más cables y se podria decir que de una forma más ordenada si los cables dupoint fuesen a la medida como se ve en la imagen, aunque en la realidad es todo un nudo de cables y lo veran más adelante.

~ Practica 8 ~

Para la Practica 8 utilizaremos 2 leds, 2 resistencias, 1 protoboard, cables dupoint y el Arduino Uno, ahora haremos parpadear a dos leds simultáneamente.

Programa

para la programación tenemos solamente un pinMode, siendo el pinmode 13, al cual simplemente le estamos dando el valor de HIGH y low para crear un parpadeo, el codigo es el siguiente:

void setup()

{

pinMode(13, OUTPUT);

}

void loop()

{

digitalWrite(13, HIGH);

delay(1000);

digitalWrite(13, LOW);

delay(1000);

}

Diseño fisico

Ahora en las conexiones fisicas, manteniendo el mismo sistema de antes, ahora conectamos otra resistencia a la columna donde llega el pin13 y otro LED al que le llega la resistencia y el gnd de la misma protoboard.

Construcción

Siguiendo la estructura anterior ahora se conecta un cable dupoint de la resistencia a otra para que siga teniendo la señal del pin13, la cual se conecta al catodo de otro led que recibira igualmente un cable dupoint que lo conecte al gnd.

Resultados

Podemos ver como ahora son los dos leds, que estan conectados al mismo pin, parpadear con un retraso de un segundo de manera simultánea.
